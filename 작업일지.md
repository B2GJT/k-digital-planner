### 단계 순서 (현재)
1. 기본 정보
2. 시장조사 (AI 자동생성)
3. 타겟 페르소나 (AI 자동생성)
4. 구성 방향
5. Edge 후보 (AI 자동생성)
6. 기획안 초안 출력

### 기획안 출력 섹션
- 00. 기본 정보
- 01. 시장 조사 (불렛포인트)
- 02. 타겟 페르소나 (페인포인트 1인칭 구어체, 불렛포인트)
- 03. 상세페이지 구성 방향 (Edge 불렛포인트, 과정명 후보 3개)

- `vercel.json`에서 `nodejs20.x` runtime 형식 오류 → functions 블록 삭제로 해결

- git 커밋 author 이메일이 `kyeongmin@k-digital-planner.local`로 잘못 설정되어 Vercel 배포 block → 로컬 git config 수정으로 해결

### 기능 추가
- 타겟 페르소나 **AI 자동생성** 버튼 추가 (`suggest-persona.js` API 신설)
- Edge 후보 **AI 자동생성** 버튼 추가 (`suggest-edges.js` API 신설)
- 단계 순서 변경: 기본정보 → 시장조사 → 페르소나 → 구성방향 → Edge → 기획안

### UI 개선
- 시장조사·페르소나·Edge 출력 전부 불렛포인트 형태로 변경
- AI 프롬프트에 `"- 항목\n- 항목"` 형식 강제하여 불렛 렌더링 보장
