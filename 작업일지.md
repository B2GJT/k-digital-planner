### 기획안 출력 섹션

## 내일 해야 할 것

### 1. Vercel 배포 마무리 (10분)
1. vercel.com → Continue with GitHub 로그인
2. Add New Project → `k-digital-planner` Import
3. Environment Variables 에 아래 값 입력:
   - `GROQ_API_KEY` = Groq 콘솔에서 새로 발급받은 키
4. Deploy 클릭
5. 생성된 URL 팀원 공유

### 2. 테스트
- 실제 과정명으로 Step 1~6 전체 흐름 테스트
- 기획안 출력 결과 품질 확인

## 중요 메모

- **Groq API 키**: 오늘 대화에 노출됐으므로 반드시 재발급 필요
  → https://console.groq.com → API Keys → 기존 키 삭제 후 새로 발급
- API 키는 절대 코드에 넣지 말고 Vercel 환경변수에만 입력
- 로컬 개발 필요할 때: 핸드폰 핫스팟으로 `npm install` 한 번 실행
