AOP 페이지는 이 기획 과정의 원본 참고 콘텐츠입니다.
AOP 과정명·타겟과 기획 과정의 불일치는 당연한 것이므로 문제로 언급하지 마세요.
기획서에 도움이 되는 내용만 추출하고, 무관한 내용은 버리세요.

---

## 1. AOP 강의 정보 (CE 04. AOP 강의 항목용)

- AOP 강의명
- 기획 과정과 흐름·목표가 유사한 파트 또는 구간 (재활용 가능한 것)
- 형식 예시: `[AOP 강의명] — [기획 과정과 연결되는 파트/구간]`

## 2. 강사 정보 (CE 04. 강사 항목용)

- AOP에서 확인된 강사명, 현직, 전직 경력
- AOP에서 확인 불가 시 "AOP 미확인"으로 표시 (inputs.txt에 있으면 참고)

## 3. 마케팅 메시지·셀링 포인트 (Edge·페르소나 보강용)

- AOP 상세페이지에서 강조된 핵심 메시지, 카피, 차별점
- 기획 과정 방향과 유사하여 재사용·재구성 가능한 문구
- 타겟 수강생 묘사 방식 (페르소나 구체화에 활용)

## 4. 사용 툴·버전 정보 (CE 04. 사용 툴 및 주의사항용)

- AOP 페이지에 명시된 사용 툴, 소프트웨어, 버전 요구사항
- 확인 불가 시 "AOP 미확인"으로 표시
