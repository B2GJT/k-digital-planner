CR 추출 내용은 강의의 실제 구성과 전달 범위를 파악하기 위한 자료입니다.
파트명과 클립명을 중심으로, 이 과정이 어떤 순서로 무엇을 가르치는지 분석하세요.

반드시 아래 제목을 유지하세요.

## 1. 러닝타임
- CR에서 총 러닝타임, 차시 수, 클립 수, 분량을 확인할 수 있으면 정리하세요.
- 확인되지 않으면 "확인 필요"로 표시하세요.

## 2. 커리큘럼 구조
- 파트별로 핵심 주제를 1줄로 요약하세요.
- 각 파트가 어떤 학습 흐름을 만드는지 설명하세요.
- 단순 클립명 나열이 아니라 "입문 -> 도구 활용 -> 실습 -> 산출물"처럼 구조를 해석하세요.

## 3. 핵심 학습 목표
- 수강생이 과정 이후 할 수 있게 되는 행동을 5개 내외로 정리하세요.
- 가능하면 "이해한다"보다 "작성한다", "분석한다", "설계한다", "자동화한다"처럼 행동 동사를 쓰세요.

## 4. 기획안 반영 포인트
- 상세페이지, 홍보 문구, 커리큘럼 소개에 반드시 반영해야 할 포인트를 정리하세요.
- CR 기준으로 강점이 될 수 있는 실습, 산출물, 도구, 차별화 요소를 뽑으세요.
- CR만으로 확인되지 않는 내용은 "검증 필요"로 표시하세요.

## 5. 주의/검증 필요 사항
- 누락된 러닝타임, 중복 파트, 불명확한 클립명, 과정 흐름상 어색한 부분이 있으면 적으세요.
